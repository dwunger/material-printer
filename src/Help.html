QC Material Label Printer - Help

? Show this help output
Opens this help page in your browser with more detailed information about the available commands.

User Controls

b Printer Downtime Barcodes
Launches the barcode generator tool to print barcodes during EPIC downtime situations.
e Print ISE Calibration labels
Prints labels for the ISE calibrators.

1. Select a temporary printer (Suggested to use the split-label printer in Micro)

2. Scan the QR codes on that boxes the calibrators came in.
- If the box is unavailable. Press the end key and enter the information manually.

3. Enter the number of copies to print and press enter.
f Flush printing queue
To conserve resources, labels print only in pairs. If you only need to print one label, you may flush the queue with the `f` key.

The side pane will show if there exist any pending print jobs in the queue.

Pressing `f` has no effect if nothing is in the print queue.

h Show help output
Pushes control mappings into the side pane.
o Toggle material open status
Toggles whether material/vials should be treated as opened or unopened. This affects how the label is formatted and expirations are calculated.
p Select printer
Opens the printer selection dialog to choose which printer to use for label printing. Your printer preference is saved. Updates will, however, overwrite your configurations.
r Edit online resources
Opens the Google Sheets configuration file in your browser to edit available online resources.
x Print image
Reads in image from user's desktop named "pepe.png," applies 1-bit dithering, and formats into ZPL command to stream into printer's bitmap buffer. Name an image on desktop "pepe.png" and press `x` to print image to selected printer.

Developer Controls

c Open PowerShell ISE
Opens the PowerShell Integrated Scripting Environment (ISE) with the application script loaded for editing and debugging.
d Start Runtime Debugging
Enters a nested PowerShell prompt for runtime debugging and troubleshooting of the application.
F5 Reload application
Completely reloads the application, refreshing all modules and configuration files to apply any changes.
m Muginn utility
Launches the Muginn utility function for configuring label printers by name.
u Force update
Forces an update check and installs any available updates for the application.
